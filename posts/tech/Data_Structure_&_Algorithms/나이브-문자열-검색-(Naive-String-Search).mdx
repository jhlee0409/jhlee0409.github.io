# 나이브 문자열 검색 (Naive String Search)

- 긴 문자열에서 부분 문자열을 검색하는 것과 관련이 있다.
- 긴 문자열에서 짧은 문자열이 등장하는 횟수를 세야하는 상황에서 쓰인다.
- 간단한 접근법 중 하나는 문자 하나씩 확인하는 것이다.

&nbsp;

# 나이브 문자열 검색의 의사코드 (Pseudocode)

- 긴 문자열 1개와 짧은 문자열(찾으려는 문자열) 1개를 인자로 받는 함수를 작성한다.
- 긴 문자열의 각 문자를 반복하는 루프를 작성하고 그 안에 짧은 문자열을 반복하는 루프도 하나 작성한다.
- 문자가 일치하지 않으면 내부 루프를 벗어난다. (루프를 멈춘다)
- 문자가 일치하면 계속 진행한다.
- 만약 내부 루프를 완료하고 일치하는 문자열을 찾았으면 `count`를 증가시키고 마지막에 그 `count`를 반환한다.
- 일치하는 문자열이 하나도 없으면 `0` 반환한다.
- 긴 문자열을 반복하는 루프를 만드는 데, 그 루프 안에는 짧은 문자열을 반복하는 중첩(nested loop)가 있다.
- 짧은 문자열의 끝에 도달했는데 아직 루프에서 벗어나지 못했으면 `count`를 `1`을 더한다.
- 루프를 벗어나지 못했다는 뜻은 일치하는 짧은 문자열과 일치하는 문자열을 찾았다는 뜻이다.

&nbsp;

## 나이브 문자열 검색 구현

```jsx
const naiveSearch = (long, short) => {
    let count = 0;
    for (let i = 0; i < long.length; i++) {
        for (let j = 0; j < short.length; j++) {
            if (short[j] !== long[i + j]) break;
            if (j === short.length - 1) count++;
        }
    }
    return count;
};

naiveSearch("lorie loled", "lol"); // 1
```
