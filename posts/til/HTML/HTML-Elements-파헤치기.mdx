# HTML Element란?

- 웹 페이지를 나타내기 위한 `HTML`의 구성 요소이다.
- `Start tag` 부터 시작해서 `End tag` 를 포함한 전부를 나타낸다.

# HTML Element 구성

![HTML Element의 구성 요소](/assets/posts/til/HTML/HTML-Elements-파헤치기/01.png)

1. 각 `Tag name` 이 가지고 있는 의미가 있다.
2. `Tag name` 은 대소문자를 구분하지 않지만 `W3C`에서는 `소문자`를 권장하고 있으며 `XHTML`과 같은 더 엄격한 문서에서는 `소문자`를 요구한다.
3. `Attribute(속성)` 은 각 `Element` 의 `Start Tag` 에 입력하며 `Element`마다 가지고 있는 고유의 속성과 공통적인 속성이 있으며, 각 `Element` 의 추가적인 정보를 제공한다.
4. `Content` 는 `Element` 별로 있을 수도 없을 수도 있다.
5. 필히 `Element` 는 `Start tag` 로 시작해서 `End tag` 로 끝을 맺어줘야 한다.
6. `Element` 는 중첩이 가능하다.
