# export 파헤치기

- 지난 포스팅에 이어서 오늘은 `export`에 대해 알아보자
- `export` 할 모듈은 또한 무조건 `엄격 모드`
- HTML 안에 작성한 스크립트에는 `import` 사용 불가


### 사용법
1. 기본 내보내기
- 한 모듈에 하나만 내보내기 가능
```javascript
// 선언식
export default function() {};
export default function*() {};
export default class {};

// 표현식
export default function name() {};
export default function* name() {};
export default class name {};

```
1. 1 import 시에는
```javascript
// 어떤식으로 네이밍 하던 상관없음, 하나만 export니까!
import anyName from 'export-default-module.js'

// if function
anyName()
// if const, let, var
anyName
```

2. 하나 or 여러개 내보내기
```javascript
// 하나씩
export const name1 ...;
export let name2... ;
export var name3...;
export function name4() {};
export class name5{}

// 여려개
export {name1, name2, name3, name4};

// with alias
export { name1 as aliasName, ...};
```
